# Infection First Line If Penacillin allergic
1 Meningitis Ceftriaxone IV 2g BD. Add Amox IV 2g QDS if age >55 or immunocompromised. Add acyclovir IV (10mg/kg) if encephalitis suspected. Chloramphenicol IV 25mg/kg QDS. Add Co-trimoxazole 1.44g BD if age >55 or immunocompromised.
2 Community Acquired Pneumonia Mild:Amox PO 1g TDS. Moderate:Doxy PO 200mg OD on day 1, then 100mg OD. Severe:Ben Pen IV 2.4g QDS + Clarithromycin IV 500mg BD Mild:Doxy PO 200mg OD on day 1, then 100mg OD. Moderate:No change.
